About the Developer

Hi! I'm Kajal, a student web developer passionate about creating clean, user-friendly websites that serve a purpose. I developed the WestCare Medical Clinic website as part of an academic project to showcase both technical skills and thoughtful design for real-world applications.

This website was built using HTML, CSS, and JavaScript to ensure a responsive and accessible user experience. I implemented structured navigation, interactive features, and modern layout techniques to provide visitors with intuitive access to information about the clinic and contact details.

I focused on maintaining consistency across pages, optimizing for different screen sizes, and using external libraries like Chart.js to visualize key clinic data. The site includes multiple pages with consistent navigation, a mock CRUD form, and a Chart.js-based data visualization.
